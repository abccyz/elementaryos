## Elementary OS Config
>普通用户使用elementary_config即可

######一键使用命令：
``` shell
git clone https://github.com/ysfl/elementaryos.git && ./elementaryos/elementary_config
```

###如果您已下载
``` shell
cd "You elementaryos File Dir"
sudo ./elementary_config
```
